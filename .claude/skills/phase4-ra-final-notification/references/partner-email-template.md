# 파트너 승인 요청 이메일 템플릿

형식: plain text (기본)

## 템플릿

```
파트너님 안녕하세요, 김태현 입니다.

팀에서 {근무시작일 (요일)} 근무 시작 예정으로 RA {N}분 채용 진행하고자 합니다.
승인해주시면 프로세스 따라 RA 채용 진행하도록 하겠습니다.

바쁘신 와중에도 항상 팀 배려해주시고 신경써 주셔서 감사드립니다. 🙂

{이름}
• {대학교} {전공} ({학점 X.XX/Y.YY})
• {핵심 경력 1}
• {핵심 경력 2}
• {핵심 경력 3}

{다음 후보자 반복}
```

## 치환 규칙

- `{근무시작일 (요일)}` → `project_settings.json`의 `work_start`에서 변환 (예: "2/27 (금)")
- `{N}분` → 합격자 수
- 후보자 프로필: 이름 + 불렛(•) 형태로 학교/전공/학점 + 경력 3줄
- 핵심 경력 요약:
  - Line 1: 컨설팅/RA 경험 (있으면)
  - Line 2: 주요 인턴 경력
  - Line 3: 기타 관련 경험 (학회, 동아리 등)
