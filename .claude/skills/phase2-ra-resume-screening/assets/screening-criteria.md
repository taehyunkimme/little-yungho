# 이력서 스크리닝 기준

## Stage 1: 자격 필터 (Pass/Fail)

### 1. 타겟 대학
- `references/target-universities.md` 참조
- 국내: 서울대, 연세대, 고려대, KAIST, POSTECH (5개)
- 해외: 글로벌 Top 30 (중국 대학 제외)
- 목록에 없는 대학은 탈락

### 2. 학년
- 대학 입학년도 기준 3학년 이상 (`현재연도 - 입학연도 >= 2`)
- 입학년도를 확인할 수 없는 경우 통과 처리

### 3. 나이
- 첫 대학 입학년도 기준으로 만 나이 추정
- 현재연도 - 입학연도 > 7이면 만 26세 초과로 판단하여 탈락
- 확인 불가 시 통과 처리

### 4. 근무 가능 기간
- 이메일 제목에서 파싱한 근무 가능 기간이 프로젝트 기간을 커버하는지 확인
- 기간 미충족자가 있을 경우 즉시 탈락시키지 않고, 집계 후 사용자에게 결정을 요청
- 기간 미기재 시에도 미충족으로 분류하여 사용자 결정에 포함

## Stage 2: 종합 평가 (총 45점 만점)

### 1. 리서치/분석 경험 (0-30점) — Tier 기반

`references/ra-firms.json`을 참조하여 경험을 Tier로 분류합니다.

| Tier | 점수 | 해당 경험 |
|------|------|----------|
| Tier 1 | 25-30점 | MBB (BCG, McKinsey, Bain) RA 경험 |
| Tier 2 | 15-20점 | PE/VC 인턴, IB 인턴, 기타 전략컨설팅(Kearney, Roland Berger, Oliver Wyman 등) RA/인턴, Big4 전략부문(Monitor Deloitte, EY-Parthenon, Strategy&) 인턴 |
| Tier 3 | 5-10점 | 기업 전략/기획팀 인턴, 씽크탱크/리서치기관, 학교 연구실 RA (경제/경영/공학 관련) |
| 없음 | 0점 | 리서치/분석 관련 경험 없음 |

- 동일 Tier 내 복수 경험 시 상위 점수 적용 (중복 합산 아님)
- 상위 Tier 경험이 있으면 하위 Tier는 무시

### 2. 스타트업/주도적 경험 (0-10점)

| 점수 | 해당 경험 |
|------|----------|
| 8-10점 | 직접 창업 + 의미 있는 traction (매출, 투자유치, 유명 액셀러레이터 선발 등) |
| 5-7점 | 직접 창업 경험 (초기 단계 포함) |
| 3-4점 | 창업 동아리/공모전에서 실질적 사업 운영 경험 |
| 0점 | 해당 없음 |

### 3. 프로젝트 관련성 (0-5점)

- 프로젝트 주제(`project_topic`)와 연관 배경(학과, 인턴, 수강과목 등): 5점
- 간접적 관련성: 2-3점
- 없음: 0점
