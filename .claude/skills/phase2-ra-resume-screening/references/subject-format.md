# 이메일 제목 파싱 형식

## 표준 형식

```
RA지원_학교명_이름_MMDD-MMDD
```

### 예시

- `RA지원_서울대_홍길동_0330-0525` → 학교: 서울대, 이름: 홍길동, 기간: 03/30~05/25
- `RA지원_연세대학교_김철수_0227-0403` → 학교: 연세대학교, 이름: 김철수, 기간: 02/27~04/03
- `RA지원_KAIST_이영희_0301-0430` → 학교: KAIST, 이름: 이영희, 기간: 03/01~04/30

## 변형 처리

### 구분자 변형
- 언더스코어(`_`), 공백(` `), 하이픈(`-`) 모두 구분자로 인식
- 예: `RA지원 서울대 홍길동 0330-0525`도 유효

### 날짜 구분자 변형
- 하이픈(`-`), 물결표(`~`) 모두 기간 구분자로 인식
- 예: `0330~0525`도 유효

### 날짜 없는 경우
- `RA지원_서울대_홍길동` → 기간 미기재로 처리 (Stage 1 기간 검증 탈락)

### 연도 경계 처리
- 종료일이 시작일보다 이전이면 다음 해로 처리
- 예: `1201-0228` → 12/01 ~ 다음해 02/28
